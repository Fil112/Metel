# ❄️ Metel (Метель)

**Metel** — это хардкорный атмосферный плагин для серверов Minecraft, который погружает игроков в суровые условия вечной зимы. Плагин полностью меняет генерацию биомов, ограничивает видимость плотным туманом, вводит механику обморожения и добавляет локальный чат для большего реализма.

**Поддерживаемые версии:** `1.16.5` — `26.2`

---

## ✨ Основные возможности

### 🥶 Механика замерзания
Игроки должны выживать в суровых холодах:
* Если игрок находится на открытом воздухе (под открытым небом), он начинает замерзать.
* **Симптомы:** Сначала накладывается эффект замедления, затем игрок начинает получать периодический урон.
* **Как согреться:** * Подойти к источникам тепла (костры, печи, факелы, лава и т.д.). Радиус действия настраивается для каждого блока индивидуально.
    * Надеть **полный сет кожаной брони**. Броня защищает от холода, но со временем изнашивается и получает урон от мороза.

### 🌫️ Видимость и Туман
Плагин создает гнетущую атмосферу неизвестности:
* Принудительно ограничивает дальность прорисовки чанков (настраиваемый туман).
* Игроки, находящиеся за пределами видимости, скрываются на уровне сервера (их нельзя увидеть даже с помощью читов).
* Никнеймы над головами игроков скрыты по умолчанию.
* Координаты в F3 можно отключить. Вместо этого координаты отображаются в Action Bar, если взять в руки **компас**.

### 🌍 Зимний мир и биомы
* Плагин автоматически (или по команде) заменяет все стандартные биомы верхнего мира на их зимние аналоги (Тайга, Снежные равнины, Замерзший океан и т.д.).
* Вечный снегопад и отключение обычного дождя.
* Вода на открытом воздухе автоматически замерзает в лед при обновлении блоков.

### 💬 Локальный чат
* Чат ограничен по расстоянию (по умолчанию 25 блоков). Чтобы услышать друг друга, игрокам придется держаться вместе.

### 🧊 Кастомные блоки
Плагин добавляет уникальные блоки с особыми механиками:
* **Тепло-генератор:** Излучает тепло на огромные расстояния и растапливает лед/снег в заданном радиусе.
* **Ледяной камень:** Распространяет экстремальный холод, замораживает воду и наносит урон всем вокруг, накладывая замедление.

## 🛠️ Команды и Права (Permissions)

| Команда | Описание | Права (Permission) |
| :--- | :--- | :--- |
| `/metel` | Показать справку по плагину | `metel.command` |
| `/metel reload` | Перезагрузить конфигурацию | `metel.command` |
| `/metel winterize [мир]` | Преобразовать обычный мир в зимний | `metel.winterize` |
| `/metel refresh [мир]` | Принудительно обновить биомы в загруженных чанках | `metel.winterize` |

**Дополнительные права:**
* `metel.bypass` — Игрок с этим правом не замерзает, не ограничивается в дальности видимости и игнорирует эффекты плагина (выдается OP по умолчанию).
